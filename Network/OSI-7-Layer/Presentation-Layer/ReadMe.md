# Presentation Layer 란?

"Presentation Layer"는 데이터를 주고 받을 때, 데이터의 형태를 표현하는 계층이다.

표현하는 방식은 여러가지가 존재한다.

## 표현 방식

1. 암호화 방식
2. 압축 방식 (ZIP)
3. 파일 형태 (JPEG, MPEG)

## 정보 제공 형태

정보의 제공 형태는 HTTP Header에 담겨있다.
